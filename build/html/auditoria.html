Soluções de Auditoria#

Para finalizar nosso Banco de Dados, vamos criar algumas soluções de auditoria utilizando metódos de Stored Procedures, Triggers e Tabelas Temporárias.

Tabelas Temporárias#

Primeiramente precisamos criar as tabelas temporárias que irão receber os dados que depois serão alvos de auditoria.

Salas#

1create table tb_sala_auditoria (
2   operacao char not null,
3   usuario varchar not null,
4   dt_hr timestamp not null,
5   id_sala integer,
6   bloco varchar(2) constraint nn_bloco not null,
7   capacidade_max integer constraint nn_capacidade_max not null,
8   tem_projetor boolean constraint nn_tem_projetor not null
9);

Professores#

1create table tb_professor_auditoria (
2   operacao char not null,
3   usuario varchar not null,
4   dt_hr timestamp not null,
5   id_professor integer,
6   nome varchar(50) constraint nn_nome not null,
7   instituto_faculdade varchar(50) constraint nn_inst_facul not null
8);

Disciplinas#

1create table tb_disciplina_auditoria (
2   operacao char not null,
3   usuario varchar not null,
4   dt_hr timestamp not null,
5   id_disciplina integer,
6   id_professor integer constraint nn_id_professor not null,
7   nome varchar(50) constraint nn_nome not null,
8   ch integer constraint nn_ch not null
9);

Agendamentos#

1create table tb_agendamento_auditoria (
2   operacao char not null,
3   usuario varchar not null,
4   dt_hr timestamp not null,
5   id_agendamento integer,
6   id_sala integer constraint nn_id_sala not null,
7   id_disciplina integer constraint nn_id_disciplina not null,
8   horario timestamp constraint nn_horario not null
9);

Triggers e Stored Procedures#

Enfim criaremos triggers para armazenar as alterações (insert, update e delete) feitas nas tabelas.

Aviso

Os códigos de criação das funções de auditoria não estão carregando o syntax highlight. Apenas as criações das triggers em si estão coloridos de acordo.

Salas#

 1CREATE OR REPLACE FUNCTION matriz.fn_sala_auditoria()
 2RETURNS trigger AS
 3$$
 4   BEGIN
 5      IF(tg_op = 'DELETE') THEN
 6         INSERT INTO tb_sala_auditoria
 7         SELECT 'E', user, now(),OLD.*;
 8         RETURN OLD;
 9      ELSIF(tg_op = 'UPDATE') THEN
10             INSERT INTO tb_sala_auditoria
11         SELECT 'A', user, now(),OLD.*;
12             INSERT INTO tb_sala_auditoria
13         SELECT 'A', user, now(),NEW.*;
14         RETURN NEW;
15      ELSIF(tg_op = 'INSERT') THEN
16             INSERT INTO tb_sala_auditoria
17         SELECT 'I', user, now(),NEW.*;
18         RETURN NEW;
19      END IF;
20      RETURN NULL;
21   END
22$$
23LANGUAGE plpgsql;

1create trigger tg_sala_auditoria
2after insert or update or delete on matriz.tb_sala
3for each row execute procedure matriz.fn_sala_auditoria();
